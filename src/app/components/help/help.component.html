Preguntas Frecuentes


¿Cómo vinculo la cuenta de mi hijo?

Para vincular la cuenta de tu hijo basta con ir a “Mi cuenta”. En la parte inferior del recuadro aparecerá un botón de “Vincular hijo”, una vez le des click aparecerá un código temporal de 5 minutos para que puedas acceder desde el celular o dispotivo de tu hijo que desees vincular. Para ello debes tener al aplicativo móvil antes.
Si la vinculación fue exitosa te aparecerá un mensaje donde estaba el código diciendo “vinculación exitosa”

¿Cómo desbloqueo una aplicacion que bloqueé por error?

Para desbloquear una aplicación debes ir a la ventana del hijo que desees desbloquear la app, por defecto verás la lista de aplicaciones bloqueadas. Debes ubicar la app que desees desbloquear en la lista y, en la misma fila, seleccionar el icon para eliminar el cual es un tacho de basura color rojo. Una vez le des click te aparecerá una ventana emergente para confirmar que deseas desbloquear esa app.
Una vez le des a “Aceptar” la aplicación desaparecerá de la lista de aplicaciones bloqueadas y podrás usarla cuando desees.
Si le das a “Cancelar” no se producirá ningún cambio

¿Cuántas cuentas puedo vincular?

Es muy frecuente la duda sobre cuantas cuentas puedo administrar para que mis hijos no se distraigan. Uno puede vincular hasta 5 cuentas de hijos para supervisar y añadir distintos dispositivos por cada uno, siempre y cuando el dispositivo esté en la lista de dispositivos de la plataforma.

¿Cómo bloqueo una aplicación?

Para bloquear una aplicación, primero debes estar en la página del hijo que le deseas bloquear la app, una vez ahí se situará por defecto en la lista de apps ya bloqueadas. Seleccionas el dispositivo en el que deseas que se bloquee. Una vez seleccionado, en la parte inferior izquierda habrá un botón de “+” donde podrás bloquear otra app.
Cuando le des click aparecerá una ventana emergente con el listado de apps, solo tendrás que seleccionar la que desees y aparecerá otra ventana con los días que deseas que se bloquee. Una vez configurado todo ello le das a “Aceptar” y verás que la app se agregó a la lista que estaba al inicio.
